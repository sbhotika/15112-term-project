--> Software needed:

Python 3 (with inbuilt libraries: tkinter, random, math)
An interpreter to compile and run code (eg: Sublime, Idle, IEP)

This can be found at: https://www.python.org/downloads/release/python-351/


--> What my project is:

Random generation of mazes using Kruskal's algorithm or Prim's algorithm 



--> How to run:

Main file: user_main.py (run on Python interpreter)
Auxillary files: Maze.py, kruskal.py, prim.py, player.py
